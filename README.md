# Nest-React-Template

This is a Nest + Next JS template.

Features:

- NestJS
- NextJS
- SCSS
- Tailwind
- Husky + Commitizen
- Jest
- MobX + React Service Provider
- Isomorphic Rendering
- Enviromnent Folder (scr/env/(dev,prod,test))

## Environments

To configure the environment you need to configure different environments at src/env

## Production

`npm run build`

`npm start`

## Development

`npm run start:dev`

## Testing

`npm run test`

or with majestic

`npx majestic --noOpen --port 5000`

## Testing Watch

`npm run test:watch`

## Testing Coverage

`npm run test:cov`

## Git Commit

`npm run commit`

## Release

`npm run semantic-release`
